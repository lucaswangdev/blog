how to use vue

兼容性

Vue.js 不支持 IE8 及其以下版本，因为 Vue.js 使用了 IE8 不能模拟的 ECMAScript 5 特性。 Vue.js 支持所有兼容 ECMAScript 5 的浏览器。

安装

  1. Vue Devtools(Chrome扩展的vue调试工具)
  2. 下载vue.js文件或者使用npm安装
